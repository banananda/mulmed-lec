Merge Sort
Merge Sort is a Divide and Conquer algorithm. It divides the input array into two halves, calls itself for the two halves, and then it merges the two sorted halves
Bubble Sort
The simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in the wrong order
Bogosort
Not necessarily useful for sorting, but may be used for educational purposes, to contrast it with more efficient algorithms
